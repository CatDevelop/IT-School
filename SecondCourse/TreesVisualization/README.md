# Выпускная работа за второй курс обучения - визуализатор структуры данных "Деревья"
**Доступные возможности**:
* Создание бинарных и AVL деревьев
* Вставка нового элемента
* Поиск элемента
* Удаление элемента
* Обход дерева
* Проверка на симметричность
